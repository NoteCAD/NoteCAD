NoteCAD Manual

Parametric CAD Sketcher — version 0.5  |  NoteCAD.online

Overview

NoteCAD is an open-source, parametric 2-D/3-D CAD application built on the Unity engine. It features a fully custom geometric constraint solver written in C# that lets you create precise, dimension-driven sketches and then convert them into solid bodies through Extrude and Revolve operations.

Try it in your browser at http://NoteCAD.online — no installation needed.

Sketch Entities

Points, segments, arcs, circles, ellipses, rectangles, polygons, splines, function curves, text, and offset entities.

Geometric Constraints

Coincident, horizontal/vertical, parallel, perpendicular, tangent, equal, fixed, point-on, distance, length, angle, mid-point, and equation constraints.

3-D Features

Workplane definition, Extrusion, and Revolve. Produces solid meshes you can export as STL or G-code.

Import / Export

Import DXF, STL, and SolveSpace files. Export STL solids and G-code tool-paths.

Drawing Entities

All drawing tools are located in the top toolbar. Select a tool, then click (and optionally drag) on the canvas to create the entity. Press Escape or right-click to finish placing a tool.

When drawing dimensions is enabled (see Settings), a dimension is automatically created as you draw. You can type a numeric value immediately after clicking to fix the dimension, or press Tab to switch between the length and angle dimension inputs.

Point

Point

Creates a free reference point on the sketch. Useful as a handle for constraints or as an anchor in Workplane / Revolve operations.

How to use: click once on the canvas to place the point.

Segment

Line Segment

Draws straight line segments. Consecutive clicks chain segments together (the end-point of the previous segment becomes the start-point of the next).

How to use: click the start point, then click the end point. Continue clicking to chain segments, or press Escape to stop.

While drawing, the tool snaps to horizontal, vertical, parallel, and perpendicular alignments. Hold Ctrl to constrain direction to 45° increments.

Arc

Arc

Creates a circular arc defined by two end-points. The center is set automatically midway and can be dragged afterwards.

How to use: click the first end-point, move to define the arc, click the second end-point.

Circle

Circle

Creates a full circle. The filled green circles visible in the screenshot are closed circles with a solid fill style applied.

How to use: click the center point, move to set the radius, click to confirm.

Ellipse

Ellipse

Creates an ellipse with independent horizontal and vertical radii. The large green tilted ellipse in the screenshot is an example.

How to use: click the center point, move to set both radii simultaneously (X and Y half-axes correspond to mouse X/Y offset from center), click to confirm.

Rectangle

Rectangle

Creates an axis-aligned rectangle (four connected line segments). The small green square in the screenshot is a rectangle.

How to use: click one corner, move to the opposite corner, click to confirm.

Polygon

Regular Polygon

Creates a regular n-sided polygon. The green hexagon with dimension labels in the screenshot is a polygon. The number of sides can be set in the tool options.

How to use: click the center, move to control size, click to confirm.

Spline[B]

Cubic Bézier Spline

Creates a smooth cubic Bézier spline through a series of clicked points. The wavy curve shown in the upper-right area of the screenshot is a spline.

How to use: click to add control points; press Escape or right-click to finish the spline.

Function

Function Curve

Creates a curve where X and Y are defined by user-supplied mathematical expressions (parametric equations). The wavy oscillating curve in the screenshot is a function curve.

How to use: click the start and end points to define the parameter range end-points; enter expressions in the dialog.

Text

Text Entity

Places a decorative text string on the sketch (as shown by "Text" in the screenshot). Text entities are sketch objects and can be moved and constrained like other entities.

How to use: type the desired text in the tool options panel, then click on the canvas to place it.

Offset

Offset

Creates an offset copy of an existing entity (line, arc, circle, etc.) at a specified distance.

How to use: click an entity to create an offset; the offset distance can be controlled by the constraint applied automatically.

Constraints

Constraints are found in the left sidebar. They restrict the degrees of freedom (DOF) of sketch entities. The status bar shows the current DOF count — a fully constrained sketch has DOF: 0.

After applying a dimension constraint (Length, Dim, Angle, Equation), its value appears as a label on the canvas. Double-click the label while the Move tool is active to edit the value numerically.

Editing & Selection

Move Tool (default)

The Move tool (cursor icon, selected by default) is the primary interaction tool. It lets you:

  • Select entities and constraints by clicking them.
  • Multi-select by holding Shift while clicking, or by dragging a rectangular marquee on empty canvas space. Dragging left-to-right selects only entities fully inside the rectangle; dragging right-to-left also selects entities that cross the boundary.
  • Move entities by holding the left mouse button and dragging. The solver runs in real-time to maintain all constraints while you drag.
  • Edit dimension values by double-clicking a dimension label; type the new value and press Enter.

Delete

Select one or more entities or constraints, then click Delete in the toolbar (or press Del) to remove them.

Copy & Paste

Select entities, press Ctrl+C to copy, and Ctrl+V to paste a duplicate at a slight offset. All constraints within the selection are preserved.

Undo / Redo

Press Ctrl+Z to undo the last action, Ctrl+Y to redo it. The status bar shows how many undo steps are available and the total undo buffer size.

Trim

The Trim tool (scissors icon in the left sidebar) removes a segment of an entity between its nearest intersections with other entities. Hover over a segment to preview the part that will be removed (shown highlighted), then click to trim it.

Workplane (Wrkplne)

A Workplane defines a flat surface in 3-D space on which a new sketch can be drawn. You must create a Workplane before drawing sketch entities in 3-D mode or before using Extrude / Revolve.

  1. Click Wrkplne in the toolbar.
  2. Click a point on the canvas to set the plane's origin.
  3. Click a line to define the first axis direction (U direction).
  4. Click a second, non-parallel line to define the second axis (V direction).

NoteCAD automatically orients the plane to face the camera. After creation the camera animates to look straight at the new workplane for comfortable 2-D sketching.

The Sketch label shown in the left properties panel corresponds to the currently active SketchFeature (workplane + sketch). You can rename it and toggle Solve Parent to include parent geometry in the constraint solve.

Extrude

Extrusion sweeps the closed contours of the current sketch along the workplane's normal direction to create a solid body.

  1. Draw and fully close the sketch contours you want to extrude (circles, rectangles, polygons, or closed chains of lines/arcs).
  2. Make sure you are in the correct Workplane (the sketch feature must be active).
  3. Click Extrude in the toolbar.
  4. A new Extrusion feature is created and activated. Use the Move tool to drag the extrusion depth handle, or double-click the depth dimension to type a value.
Important: Only closed contours are extruded. Open chains of lines or arcs will not produce a solid body.

Revolve

Revolve sweeps the closed contours of the current sketch around a user-defined axis.

  1. Draw the profile you want to revolve in the active sketch. The profile should be on one side of the intended axis.
  2. Click Revolve in the toolbar.
  3. Click a point to define the origin of the axis.
  4. Click a line to define the direction of the axis.

A Revolve feature is created and activated. The revolution angle can be set in the detail editor panel on the right.

Status Bar

The status bar at the bottom of the screen provides live feedback:

FieldMeaning
Tool hint (left) Instructions for the currently active tool (e.g. "MoveTool [Escape]: hover over an entity…").
Memory (right) Current Unity memory usage (e.g. "29.85 mb").
Solver status OKAY — all constraints are satisfied. Any other message indicates a solver conflict or unsatisfied constraint.
DOF Degrees of Freedom remaining in the sketch. DOF: 0 means the sketch is fully constrained. A positive number means under-constrained; entities may still move.

Tips & Tricks

Creating a Solid Body from Scratch

  1. Start a new document (New).
  2. Draw the outline of your part using closed contours (rectangles, circles, polygons, or chained lines/arcs). Make sure every contour is closed — all end-points must be coincident.
  3. Add dimensions and constraints until DOF: 0 is shown in the status bar.
  4. Click Extrude to create the solid. Adjust the depth.
  5. To add a feature on a face, use Wrkplne to define a new sketch plane on that face, draw the new profile, and apply another Extrude or Revolve.
  6. Export the finished model as STL with Export.

Fully Constraining a Sketch

  • Start by applying a Fixation to one anchor point so the sketch does not drift.
  • Use Length to fix edge lengths, Angle for orientations, and Horizontal/Vertical for axis-aligned segments.
  • Watch the DOF counter decrease as you add constraints. When it reaches 0, all geometry is grey/green (fully constrained).

Working with Styles

The top-right panel lets you create visual styles (line color, fill, weight) and assign them to entities. Use Add to create a style, Edit to modify it, Assign to apply it to selected entities, and Remove to delete it.

The Solid dropdown sets the overall rendering mode of 3-D bodies in the viewport.

Snapping While Drawing Lines

The Line Segment tool automatically detects and snaps to:

  • Horizontal and vertical alignments.
  • Directions parallel or perpendicular to the previous segment.
  • Directions parallel to other existing lines in the sketch.
  • Point-to-point vertical/horizontal alignment (shown with a dashed guide).

Snapped constraints are applied permanently when you click. To prevent a constraint from being applied, move the mouse away from the snap direction before clicking.

Editing Constraints After Creation

Switch to the Move tool and double-click any dimension label (length, angle, distance) to open an inline text field. Type the new value and press Enter. The solver updates all constrained geometry immediately.
